# AI Student Chatbot 🤖

An AI chatbot for students built using:

- Streamlit
- Groq API
- LLaMA 3 model

## Features

- Conversational AI
- Remembers chat history
- Fast responses using Groq
- Simple UI

## Live Demo

https://ai-student-chatbot.streamlit.app

## Tech Stack

Python  
Streamlit  
Groq API  
LLaMA 3

## Run Locally

```bash
pip install -r requirements.txt
