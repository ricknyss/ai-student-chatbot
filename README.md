# AI Student Chatbot 🤖

An AI chatbot for students built using:

- Streamlit
- Groq API
- LLaMA 3 model

## Features

- Conversational AI
- Remembers chat history
- Fast responses using Groq
- Simple UI



## Tech Stack

Python  
Streamlit  
Groq API  
LLaMA 3

```bash
pip install -r requirements.txt
streamlit run app.py
